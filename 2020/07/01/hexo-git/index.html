hexo + github搭建个人博客

发表于 2020-07-01 更新于 2021-04-30

前言

作为一个不甘寂寞的程序猿，是否都想要搭建一个属于自己的个人网站，如果你曾经或者现在正有这样的想法，请跟随这篇文章，它或许能为你提供一些帮助，助你快速拥有自己的博客网站。

Hexo与github简介：


Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Heroku上，是搭建博客的首选框架之一。Hexo是GitHub上的开源项目，git地址hexojs/hexo，Hexo使用细节可到官网hexo查阅。github应该每位程序猿都知道，这里就不细说了。

本文选用Hexo + github搭建个人博客，具体步骤见下文：

搭建步骤

GitHub创建个人仓库

登录到GitHub，点击GitHub中的New repository创建新仓库，仓库名应该为：用户名.github.io 这个用户名使用你的GitHub帐号名称代替即可。例如假如git账户名为xiaomin,仓库名为xiaoming.github.io即可。

安装Git

Git是开源的分布式版本控制系统，如果对git还不胜理解的可参考廖雪峰老师的git教程。从git下载地址找到对应版本下载安装即可。安装完成后，打开命令行执行git --version查看安装版本，也可通过该命令查看是否安装成功。安装成功后配置user.name与user.email信息：

现在来介绍常用的Hexo 命令

npm install hexo -g #安装Hexo
npm update hexo -g #升级
hexo init #初始化博客
命令简写
hexo n “我的博客” == hexo new “我的博客” #新建文章
hexo g == hexo generate #生成
hexo s == hexo server #启动服务预览
hexo d == hexo deploy #部署

hexo server #Hexo会监视文件变动并自动更新，无须重启服务器
hexo server -s #静态模式
hexo server -p 5000 #更改端口
hexo server -i 192.168.1.1 #自定义 IP
hexo clean #清除缓存，若是网页正常情况下可以忽略这条命令

图床配置

博文一般为了更方便阅读，都会以图片的方式加以说明，而图片一般会占据大量的存储空间。这时考虑把博文里的图片上传到图床，然后获得外部链接，使用Markdown语法 ![图片信息](外部链接) 完成图片的插入。常见的简易的图床网站有七牛云，SM.MS(支持免费)，关于最新免费图床，自行百度即可。

个性化设置

个性化设置就是根据个人需要添加不同的插件及功能。
一些基本信息可在_config.yml中配置，如网站标题、副标题、网站描述、作者、网站头像外部链接、网站语言、时区等。
一些其他插件如微信、支付宝打赏、音乐播放器等等，这里就不一一介绍了。
